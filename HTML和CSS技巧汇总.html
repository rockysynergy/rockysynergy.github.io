文字竖排

  1. 给p或者其它包含文字的元素设置窄的宽带

替换select元素默认的下拉箭头

select {
   -webkit-appearance: none; 
   -moz-appearance: none;
   appearance: none;       /* remove default arrow */
   background-image: url(...);   /* add custom arrow */
}
