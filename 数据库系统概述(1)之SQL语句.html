数据库系统概述(1)之SQL语句

更新于：2019-02-26 23:11:43 Learn Database

第一部分对数据库系统做了概述性的总结，本文总结SQL语句。

SQL分为：

  1. 定义语句(DDL): Create database以及create table
  2. 数据修改语句(DML)：Insert, Delete, Update和Select。
  3. 控制语句(DCL): Grant以及Revoke等
